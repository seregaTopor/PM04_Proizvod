# Производственная практика ПМ.04

## Описание

Данный репозиторий содержит результаты производственной практики по профессиональному модулю **ПМ.04 «Сопровождение и обслуживание программного обеспечения компьютерных систем»**, пройденной в АО «Красный Якорь» (г. Слободской, Кировская область).

В ходе практики были выполнены работы по установке, настройке и администрированию программного обеспечения, а также по обеспечению безопасности и мониторингу IT-инфраструктуры предприятия.

---

## Выполненные задачи

В рамках практики были успешно реализованы следующие виды работ:

- **Развертывание инфраструктуры:** Установка и настройка локальной среды разработки (XAMPP), включая веб-сервер Apache, СУБД MySQL/MariaDB и интерпретатор PHP.
- **Управление доменом:** Развертывание роли Active Directory (AD DS) для централизованного управления учетными записями.
- **Веб-разработка:** Установка и настройка CMS WordPress для корпоративного портала и платформы электронного обучения Moodle.
- **Мониторинг:** Настройка системы мониторинга Zabbix для контроля состояния серверов и сетевого оборудования.
- **Криптографическая защита:** Установка КриптоПро CSP для работы с электронной подписью и государственными системами (ЕГАИС Лес, Честный Знак).
- **Автоматизация:** Реализация скриптов автоматизированного резервного копирования данных с использованием планировщика Windows.
- **Безопасность:** Настройка дискреционного управления доступом и установка антивирусного ПО.
- **Документирование:** Разработаны отчет о практике, руководство оператора и дневник в соответствии с требованиями.

Все поставленные задачи выполнены в полном объеме, система готова к эксплуатации.
